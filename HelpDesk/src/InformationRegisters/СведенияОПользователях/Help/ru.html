Для открытия списка пользователей, входящих в подразделение, следует в разделе "Нормативно-справочная информация" и с помощью команды панели навигации "Структура предприятия" открыть структуру предприятия, в списке выбрать нужное предприятие и выполнить команду "Все действия - Изменить". В карточке подразделения в панели навигации выполнить команду "Пользователи".

С помощью команд "Создать" и "Изменить" создавайте и редактируйте пользователей, входящих в подразделение. Выберите пользователя из списка пользователей, укажите его должность.

В список пользователей подразделения значение в колонке "Телефон" выбирается из карточки пользователя.
