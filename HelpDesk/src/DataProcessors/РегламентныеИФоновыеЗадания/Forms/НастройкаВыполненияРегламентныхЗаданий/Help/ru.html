В случае работы с файловой информационной базой имеется возможность настроить автоматическое выполнение регламентных заданий.

Наименование свойства А Назначение свойства
Автоматически запускать отдельный сеанс выполнения регламентных заданий +

Установка флажка настраивает режим запуска отдельного сеанса для выполнения регламентных заданий через секунду после запуска приложения.

Если приложение запущено с ключом выполнения регламентных заданий, тогда запуск будет пропущен, т.к. в качестве сеанса, обрабатывающего регламентные задания будет назначен текущий сеанс.

Отдельный сеанс запускается от имени текущего пользователя, а процедуры выполняются в привилегированном режиме.

Уведомлять о некорректном состоянии выполнения регламентных заданий Уведомлять пользователя, что регламентные задания не выполняются, если установлен флажок "Автоматически запускать отдельный сеанс выполнения регламентных заданий".
Период уведомления, минут Период в минутах, через который будет производится проверка состояния выполнения регламентных заданий с последующим уведомлением пользователя, если состояние некорректно.

Прим.: колонка "А" содержит знак "+" там, где для изменение свойства доступно только Администратору.

Команда Описание
Запустить отдельный сеанс. Только для файловой ИБ. Запускает отдельный сеанс для выполнения регламентных заданий.
Производится запуск отдельного сеанса с параметрами, такими, что сеанс открывается только для выполнения регламентных заданий и считает текущий сеанс родительским, кроме того отдельный сеанс автоматически завершается при завершении родительского сеанса.

 

Параметры инфорационной базы при запуске 1С Предприятия 8 для включения автоматического выполнения регламентных заданий.

Параметр

 Описание
DoScheduledJobs Выполнять регламентные задания в открываемом сеансе.
SkipMessageBox Не показывать предупреждения, в случах, когда сеанс, выполняющий задания, уже открыт или есть другие причины, препядствующие выполнению заданий в открываемом сеансе.
AloneIBSession Запустить сеанс только для выполнения регламентных заданий. Если сеанс для целей выполнения заданий уже открыт, то открываемый сеанс будет завершен с предупреждением (или без предупреждения см. SkipMessageBox).

Примеры строки программы в ярылке Windows.

Назначение

 Строка
Включить выполнение заданий в открываемом сеансе. "C:\Program Files\1cv82\Bin\1cv8c.exe" /CDoScheduledJobs

Запустить сеанс только для выполнения заданий. Например, если каталог информационной базы расположен на отдельном компьютере, то эту строку можно записать в планировщик Windows. Лишний запуск сеанса будет автоматически завершен.

 "C:\Program Files\1cv82\Bin\1cv8c.exe" ENTERPRISE /IBConnectionString"File=""C:\Инфорационная база"";" /NВасилий /PПароль /C"DoScheduledJobs SkipMessageBox AloneIBSession".
