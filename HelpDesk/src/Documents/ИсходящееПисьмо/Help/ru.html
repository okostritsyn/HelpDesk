Новое письмо можно создать:

  1. Из карточки входящего письма с помощью команд Ответить, Переслать.
  2. Из карточки исходящего письма с помощью команды Переслать.

Письмо также может быть создано на основании:

  • внутреннего документа,
  • входящего документа,
  • исходящего документа,
  • файла,
  • проекта,
  • проектной задачи,
  • задачи исполнителя.

В карточках писем можно работать с вложениями письма как с обычными файлами. Для входящего письма нельзя добавлять новые вложения.

При создании ответа или пересылки в письмо добавляется текст исходного письма и подпись пользователя в соответствии с настройками, указанными в форме настроек встроенной почты. Также можно добавить текст исходного письма или подпись вручную с помощью подменю Добавить в меню Все действия.

Письма могут быть связаны как между собой, так и с входящими и исходящими документами. Для этого используется общий механизм связей.

Входящее письмо автоматически формируется связь Отправлен в ответ на – Отправлено ответ.

При пересылке письма используется связь Пересылка – Переслано письмом.

При получении ответа на наше исходящее используется связь Получен ответ на – Получен ответ.

Основные связи писем отображаются в карточке письма в виде гиперссылок, например, Отправлен ответ, Получен ответ. Все связи письма можно посмотреть при помощи команды Все действия - Связи в карточке письма.

В карточке входящего письма можно написать комментарий. Это удобно использовать для того, чтобы зафиксировать дополнительные сведения, которые обязательно надо учитывать при разборе письма, например устные договоренности, предварительные соображения, ссылки и другое.

Все письма имеют реквизит Предмет. Предметом переписки может служить Внутренний документ, Входящий документ, Исходящий документ, Файл, Мероприятие, Проект, Проектная задача, Задача исполнителя. Если в письме выбран предмет, то это письмо можно будет увидеть в карточке предмета в форме переписки по предмету. При этом письма электронной почты отображаются в этом дереве совместно с входящими и исходящими документами.

При установке предмета в письме выбранный предмет автоматически устанавливается всем письмам в истории переписки с незаполненным предметом.

Реквизит Проект используется для учета трудозатрат на обработку и создание писем.

При установке проекта он будет установлен всем письма в истории переписки, у которых проект не был установлен ранее. При поступлении ответного письма ему автоматически устанавливается тот же самый проект, который был установлен исходному письму.

Учет писем в разрезе проектов используется для анализа переписки и для учета трудозатрат.
