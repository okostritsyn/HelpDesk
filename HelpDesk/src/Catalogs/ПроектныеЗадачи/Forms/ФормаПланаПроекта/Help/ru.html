На закладке Задачи проекта выводится список проектных задач, которые объединены в иерархическую структуру.

Просматривать список проектных задач можно в следующих режимах, которые отличаются набором выводимых колонок:

  • планирование – удобен для составления плана проекта,
  • отправка на выполнение – удобен для своевременной отправки задач на выполнение,
  • контроль выполнения – удобен для контроля выполнения проектных задач,
  • анализ трудозатрат – удобен для анализа плановых и фактических трудозатрат,
  • полный – выводит полный набор колонок в списке.


Редактировать и создавать задачи можно:

  • в диалоге (добавление и редактирование задачи в отдельной карточке),
  • в списке (добавление и изменение задачи в полях списка).


При редактировании в списке действует ряд ограничений:

  • нельзя указать более одного предшественника,
  • нельзя указать тип зависимости, отличный от "Окончание-начало",
  • нельзя указать более одного исполнителя,
  • нельзя изменить единицу длительности и трудозатрат,
  • нельзя изменить настройки задачи.


Полные возможности редактирования проектной задачи доступны при редактировании в карточке.

Для создания проектной задачи используется команда Добавить в списке проектных задач. При этом задача будет создана на текущем уровне иерархической структуры.

Также можно создать:

  • последующую задачу (команда Добавить последующую задачу),
  • подчиненную задачу (команда Добавить подчиненную задачу).


При помощи команд панели действия задачу можно переместить выше или ниже по списку (команды Вверх и Вниз), а также изменять иерархическую подчиненность (команды На уровень выше и На уровень ниже).

На закладке Диаграмма Ганта отображается план текущего проекта на диаграмме Ганта.

Выполненные проектные задачи отображаются на диаграмме Ганта отрезком, концы которого совпадают с фактической датой начала и окончания проектной задачи. Невыполненные поектные задачи отображаются отрезком, с концами равными плановой дате начала и окончания проектной задачи.

При помощи двойного щелчка мыши по полосе диаграммы можно открыть карточку задачи по проекту. Можно увеличить плановую длительность задачи, перетаскивая мышью отрезок задачи за край.

Отреэки диаграммы Ганта выводятся разными цветами:

  • черным - суммарные задачи (задачи, имеющая подчиненные задачи),
  • синим - задачи, которые еще не выполняются, и дата начала не посрочена,
  • коричневым - задачи, которые еще не выполняются, и дата начала просрочена,
  • зеленым - задачи, которые выполняются, и дата окончания не просрочена,
  • красным - задачи, которые выполняются, и дата окончания посрочена,
  • серым - завершенные задачи.
