Для использования встроенной почты нужно настроить учетные записи.

В карточке учетной записи обязательно необходимо указать вариант использования учетной записи Для встроенной почты. При этом станут доступны закладки Папки писем и Рабочая группа. Обязательными для заполнения также являются поля Почтовый адрес, Пароль.

При создании новой учетной записи в список ответственных автоматически добавляется текущий пользователь. При необходимости список ответственных можно расширить добавив в него всех пользователей, которые должны иметь права на чтение и отправку писем этой учетной записи.

Папки для писем создаются автоматически при сохранении учетной записи. Они будут размещены в корневой папке, имя которой будет совпадать с наименованием учетной записи. Если необходимо настроить работу с письмами разных учетных записей в одних и тех же папках, то перед сохранением карточки новой учетной записи нужно вручную указать соответствующие существующие папки писем.
