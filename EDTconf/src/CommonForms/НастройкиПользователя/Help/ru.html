Для редактирования персональных настроек можно воспользоваться командой "Демо: Мои настройки" в разделе "Настройка и администрирование" в группе "Сервис" панели действий.

Общие

Настройка Описание
Сведения о пользователе

Нажатие кнопки открывает форму настройки свойств текущего пользователя системы.

При работе с системой от лица внешнего пользователя открывается форма настройки текущего внешнего пользователя.

Установить расширение работы с файлами Требуется для выполнения определенных операций с файлами в веб-клиенте.
Настроить доступ к Интернету

Нажатие кнопки откроет форму настройки прокси сервера для доступа в сеть Интернет с данного компьютера.

Обновить параметры системы

Применяет ранее измененные для данного пользователя параметры системы. Можно использовать эту команду вместо перезагрузки программы (например, в случае если администратор изменил настройки системы, и их необходимо применить, не выходя из программы).

Настройки работы с файлами

 

Настройка Описание
Действие при выборе файла Указывает, какое действие выполнять при нажатии на клавишу Enter или двойном щелчке мыши на файле - открыть файл или открыть его карточку.
Выбор режима открытия при выборе файла Если установлена, то перед открытием файла откроется окно для выбора режима открытия - "просмотр" или "редактирование".
Показывать подсказки при редактировании файлов

Если установлена, то при работе через веб-клиент программа будет подсказывать о том, как удобнее открыть файл для просмотра или редактирования, как обновить его в информационной базе.

Показывать занятые файлы при завершении работы

Если установлена и текущий пользователь занял один или несколько файлов, то при завершении работы показывается список этих файлов. Пользователь может открыть файл для просмотра, закончить редактирование, открыть карточку файла.

Показывать колонку "Размер" в списках файлов Если включено, в форме хранилища документов и в списке редактируемых файлов показывается колонка "Размер" (в Кб).
Показывать информацию, что файл не был изменен При выполнении команды "Закончить редактирование" или "Сохранить изменения", если файл не был изменен, новая версия не создается (существующая не изменяется). При этом выдается диалог с информацией о том, что версия не была создана  - если включена настройка Показывать информацию, что файл не был изменен.
Сравнивать версии файлов ODT, DOC, RTF, TXT при помощи

Выбор одного из двух возможных вариантов сравнения файлов указанного формата - "Microsoft Office Word" или "OpenOffice.org Writer".

Настройка основного рабочего каталога Нажатие на эту кнопку откроет окно управления рабочим каталогом.

Интервал проверки напоминаний

Указывает в минутах, насколько часто необходимо проверять наступление времени напоминаний.
