С помощью настройки версионирования документов и справочников можно отслеживать историю изменений.

  • Не версионировать - история версий объекта не ведется;
  • Версионировать всегда - новая запись заносится в историю версий каждый раз при изменении (или создании нового) справочника или документа;
  • Версионировать при проведении - версии сохраняются только при проведении; применимо только для документов, для которых разрешено проведение.

Также возможна настройка версионирования сразу для группы выделенных объектов в списке.
