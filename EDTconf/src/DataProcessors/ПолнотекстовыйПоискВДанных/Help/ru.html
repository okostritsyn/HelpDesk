Полнотекстовый поиск в данных находит документы, файлы, задачи, процессы и любые другие данные не только по полям карточек, но и в содержимом файлов распространенных форматов. Например, шаблон, включающий в себя слова «Договор о поставках», принесёт результаты как из соответствующих документов информационной базы программы, так и о всех упоминаниях о «Договоре о поставках», встречающихся в иных документах.

Что делать, если нашлось слишком много данных

Если нашлось слишком много данных, есть следующие варианты решения проблемы:

  1. Уточните запрос, добавьте к нему дату, сумму, название корреспондента и.т.д.
  2. Включите расширенный поиск и укажите точные разделы поиска.

Что делать, если нашлось слишком мало данных

Если результат поиска слишком мал, есть следующие варианты решения проблемы:

  1. Упростите поисковое выражение, удалив одно или несколько слов. Не используйте кавычки. Включите дополнительно настройку Искать слова с ошибками.
  2. Используйте Обычный поиск вместо Расширенного поиска. Для этого нажмите на кнопку Скрыть расширенный поиск и повторите поиск, упростив поисковое выражение.

Советы

Для простых случаев поиска достаточно ввести одно или несколько слов.

Например: Регистрационный номер документа «126-ОГ/12» ИЛИ «Договор Цветметторг».

Для точного поиска используйте:

  • кавычки - для поиска точной фразы.Например: О задержке сроков;
  • ИЛИ - для поиска одного из двух слов.Например: Клауст ИЛИ Энергомаш;
  • скобки - для группировки выражений.Например: Петров (Клауст ИЛИ Энергомаш).

В поле ввода строки поиска можно вставить навигационную ссылку на данные. В этом случае будет сразу открыта нужная карточка. Ссылку на объект информационной базы можно получить, нажав на кнопку Получить ссылку.

Как добавить новый раздел поиска

Разделы позволяют сузить круг поискового запроса, тем самым увеличивая его точность. Можно использовать стандартные разделы или добавить свои собственные.

Чтобы создать новый раздел поиска, выполните следующие действия:

  1. Выполните щелчок правой кнопкой мыши в списке разделов.
  2. Выполните команду Настроить разделы поиска контекстного меню.
  3. В новом окне нажмите на кнопку Добавить.
  4. Дайте разделу понятное название, для этого заполните поле Наименование.
  5. Установите флажок у тех видов данных, которые хотите включить в поиск.
  6. Нажмите ОК , чтобы сохранить настройку

Общие сведения

Полнотекстовый поиск в данных ищет и по реквизитам, и по содержимому файлов распространенных форматов (по умолчанию доступны следующие форматы: ASCX, ASP, ASPX, CSS, HHC, HTA, HTM, HTML, HHT, HTW, HTX, ODC, STM, DOC, DOT, POT, PPS, PPT, XLB, XLC, XLS, XLT, TXT, EML; этот список также может быть расширен путем установки плагинов для iFilter).

Обновление индекса выполняется автоматически, но с некоторой задержкой.
