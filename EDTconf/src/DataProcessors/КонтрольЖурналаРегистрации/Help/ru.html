Для отправки отчета по критичным записям в журнале регистрации необходимо:

  • удостовериться, что настроена системная учетная запись электронной почты;

  • настроить расписание рассылки (Настройка -> Расписание регламентного задания);
  • указать адреса электронной почты получателей отчета (Настройка -> Получатели отчета).

При рассылке отчета на указанные адреса данные выбираются за период времени, прошедший с предыдущей рассылки. Например, если рассылка настроена два раза в день: в 9:00 утра и 18:00 вечера, то дважды в сутки будет рассылаться отчет, в которые включены записи журнала регистрации с 18:00 предыдущего дня по 9:00 текущего дня и с 9:00 текущего дня по 18:00 текущего дня соответственно.
