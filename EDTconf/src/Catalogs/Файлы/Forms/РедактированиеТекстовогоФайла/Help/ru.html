Как открыть файл во встроенном редакторе текстовых файлов

По умолчанию во встроенном редакторе открываются текстовые файлы с расширением txt, ini, xml. Вы можете дополнить список расширений файлов в персональных настройках программы. Для этого:

  1. Перейдите в раздел Настройка и администрирование.
  2. Выполните команду Персональные настройки командной панели.
  3. В персональных настройках программы перейдите на закладку Работа с файлами.
  4. Введите расширения файлов в колонке Расширение.
  5. Нажмите ОК, чтобы сохранить настройку.

Как открыть файл в стандартной программе

Если вы хотите, чтобы текстовые файлы c определенными расширениями по умолчанию открывались во внешнем приложении, то внесите изменения в персональные настройки программы. Для этого:

  1. Перейдите в раздел Настройка и администрирование.
  2. Выполните команду Персональные настройки панели действий.
  3. В персональных настройках программы перейдите на закладку Работа с файлами.
  4. В колонке Способ открытия выберите пункт Во внешнем приложении.
  5. Нажмите ОК, чтобы сохранить настройку.

Изменение кодировки текстового файла

Текстовые файлы могут быть сохранены в разных кодировках. Основными кодировками для работы с текстовыми файлами на русском языке являются:

  • IBM866 (Кириллица DOS),
  • KOI8-R (Кириллица KOI8-R),
  • Windows-1251 (Кириллица Windows.

Если после открытия текстового файлв вы увидели набор бессмысленных знаков, то, вероятнее всего, дело в кодировке.

Чтобы изменить кодировку текстового файла, выполните следующие действия:

  1. Откройте файл во встроенном редакторе текстовых файлов.
  2. Выполните команду Изменить кодировку в меню Все действия.
  3. Выберите кодировку из списка.
  4. Если нужная кодировка отсутствует, снимите флажок Показывать только основные кодировки.
  5. Выберите нужную кодировку из списка.
  6. Нажмите ОК, чтобы сохранить настройку

Общие сведения

Принцип работы встроенного редактора текстовых файлов такой же как и у внешнего приложения. Файл копируется на локальный компьютер и помечается как занятый в информационной базе. До тех пор пока вы не нажмете на кнопку Закончить редактирование созданная версия файла не будет доступна другим пользователям.
