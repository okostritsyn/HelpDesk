В этом списке ведется учет групп внешних пользователей.

Группы внешних пользователей могут использоваться для группировки большого числа внешних пользователей, а также когда нужно указывать не только внешнего пользователя, но и группу внешних пользователей, например, при настройке доступа.

Группы создаются плоским списком или иерархическим. В иерархическом списке, внешние пользователи вложенных групп неявно входят в группы-родители.

Группы формируются по используемому функционалу прикладного решения:

  • Контрагенты
    • Доверенные партнеры
    • Основные покупатели
    • Основные поставщики
    • Участники опроса
  • Физические лица

Существует предопределенная группа Все внешние пользователи, которая неявно включает всех внешних пользователей.

Существует возможность создать группы внешних пользователей одного типа, в том числе такие группы, которые неявно включают всех пользователей одного типа.

Когда используется непосредственное назначение ролей, тогда их можно назначать не только для внешнего пользователя, но и для группы внешних пользователей.
